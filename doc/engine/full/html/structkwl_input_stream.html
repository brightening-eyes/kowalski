Detailed Description

A struct representing an input stream, getting its data from either a file or a buffer.

Field Documentation

void* kwlInputStream::buffer

A pointer to a buffer providing the stream with data (NULL if the stream gets its data from a file).

FILE* kwlInputStream::file

A pointer to a file providing the stream with data (NULL if the stream gets its data from a buffer).

int kwlInputStream::offset

The offset in bytes into the underlying data.

int kwlInputStream::readPos

The current byte position, relative to the start of the underlying data.

int kwlInputStream::size

The size in bytes of the stream data source. A negative size indicates that the size is unknown, which is the case when reading files.
