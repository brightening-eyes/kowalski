Detailed Description

This header defines the main portion of the public Kowalski Engine API. Language bindings are generated from this header file. See kowalski_ext.h for the DSP unit API.

Enumeration Type Documentation

enum kwlDistanceAttenuationModel

An enumeration of supported positional audio distance attenuation models.

Enumerator:
KWL_INV_DISTANCE 

The gain is proportional to the inverse distance between an event and the listener.

KWL_LINEAR 

The gain decays linearly with the distance between an event and the listener.

KWL_CONSTANT 

The gain is constant and independent of the distance between an event and the listener.

Sets the user gain of an event instance corresponding to a given handle. The gain parameter passed to this method is not a linear gain factor, but a parameter that gives an approximately linear change in percieived loudness when changed linearly. void kwlEventSetLinearGain ( kwlEventHandle  handle,
float  gain 
)

Sets the user gain of an event instance corresponding to a given handle. The gain parameter passed to this method is a linear amplitude scale, where a value of 1 corresponds to unit gain and a value of 0 to complete attenuation. Note that a linear change of the gain parameter does not result in a linear change in perceived loudness (see kwlEventSetGain).

unsigned int kwlGetNumFramesMixed (  ) 

Returns the number of frames of audio mixed since the last call to this method. The accuracy of the returned value depends on the engine buffer size and this method is not meant to provide sample accurate values. However, the value is based on the actual number of mixed samples and thus functions as an approximate sample clock that is guaranteed not to drift. This makes it useful for syncing tasks where high accuracy is not crucial, for example syncing visuals to musical beats.

Error codes:

  • KWL_ENGINE_IS_NOT_INITIALIZED if the Kowalski engine has not been initialized.

void kwlSetDistanceAttenuationModel ( kwlDistanceAttenuationModel  type,
int  clamp,
float  maxDistance,
float  rolloffFactor,
float  referenceDistance 
)

Specifies a distance attenuation model to use for positional events. The role of the model parameters rolloffFactor and referenceDistance varies between different models. The gain is computed based on the distance d between the event and the listener as follows for the different models:

  • KWL_INV_DISTANCE: gain = referenceDistance / (referenceDistance + rolloffFactor * (d – referenceDistance))
  • KWL_LINEAR: gain = max(0, (1 – rolloffFactor * (d – referenceDistance) / (maxDistance – referenceDistance)))
  • KWL_CONSTANT: gain = 1.

Error codes:

  • KWL_ENGINE_IS_NOT_INITIALIZED if the Kowalski engine has not been initialized.
  • KWL_INVALID_PARAMETER_VALUE if
    • rolloffFactor is less than zero
    • referenceDistance is less than or equal to zero
    • referenceDistance is greather than or equal to maxDistance and maxDistance is greater than 0.
Parameters:
type The distance attenuation model type.
clamp Some distance models may amplify an event at small distances. Setting this parameter to a non-zero value will prevent this from happening.
maxDistance Events further away than this from the listener will be inaudible. This parameter is ignored if its value is zero or less.
rolloffFactor A parameter with a model specific interpretation. Ignored if type is KWL_CONSTANT.
referenceDistance A parameter with a model specific interpretation. Ignored if type is KWL_CONSTANT.
