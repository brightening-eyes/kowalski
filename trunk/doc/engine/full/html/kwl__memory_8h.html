#define KWL_FREE ( ptr   )     kwlFree(ptr)

This macro should be used for all memory deletions in the Kowalski Engine. If the symbol KWL_DEBUG_MEMORY is not defined, KWL_FREE reduces to a call to free.

#define KWL_MALLOC ( size,
tag   )     kwlMalloc(size)

This macro should be used for all memory allocations in the Kowalski Engine. If the symbol KWL_DEBUG_MEMORY is not defined, KWL_MALLOC reduces to a call to malloc. This macro exists both to allow for allocation tracking (if KWL_DEBUG_MEMORY is defined) and to make it easy to use custom allocators if need be. The tag parameter is a const* char const string identifying the allocation.
