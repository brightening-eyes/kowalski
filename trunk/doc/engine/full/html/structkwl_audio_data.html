Detailed Description

A structure describing a piece of audio data. In the case of PCM data, the data is just an array of signed, interleaved 16 bit samples. In the case of non-PCM data, the data may represent an entire audio file that a suitable decoder is responsible for parsing and decoding.

If this is streaming data, this indicates the byte offset into the stream of the first audio data byte.

const char* kwlAudioData::filePath

The file path relative to the Kowalski project data. Note: this has nothing to do with the runtime path, but is used to associate wave bank entries with audio data entries in the engine and is also convenient for debugging.

int kwlAudioData::numBytes

The total number of bytes of loaded audio data. A value of 0 indicates that no data is loaded.

int kwlAudioData::numChannels

The number of channels of the audio. Only used for PCM data.

int kwlAudioData::numFrames

The number of audio frames in the data. Only used for PCM data.

int kwlAudioData::streamFromDisk

Non-zero if the non-PCM data (if any) should be streamed from disk.

kwlWaveBank* kwlAudioData::waveBank

The wave bank containing this piece of audio data. NULL indicates that the data does not belong to a wavebank.
